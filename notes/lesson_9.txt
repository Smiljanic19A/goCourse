1) How do you run go code
    go run file.go - this is the command to run any go file

    the go command is part of the GO cli and these are the key commands
    ===============================================
    go build        ->          Compiles a bunch of go source code files
        The output of this command is basically a .exe file
    go run          ->          Compile and execute one or two files
    go fmt          ->          Formats all the code in each file of the current directory
    go install      ->          Compiles and installs a package
    go get          ->          Downloads the raw source code of someone elses package
    go test         ->          Run any tests associated with the current project
    ===============================================
